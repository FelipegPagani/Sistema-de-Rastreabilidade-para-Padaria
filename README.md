# 🥖 Sistema de Rastreabilidade para Padaria

## Sobre o projeto

Este projeto foi desenvolvido como trabalho final da disciplina de **Algoritmos e Programação**.

O objetivo é implementar um sistema de rastreabilidade para padarias utilizando **Java**, permitindo controlar os ingredientes recebidos, os produtos cadastrados e os lotes de produção.

## Como o sistema funciona

O sistema segue o fluxo abaixo:

1. Cadastro de ingredientes, fornecedores e clientes.
2. Registro do recebimento dos ingredientes, gerando automaticamente um lote de ingrediente.
3. Cadastro dos produtos e dos ingredientes que compõem cada receita.
4. Registro dos lotes de produção, associando os lotes de ingredientes utilizados para cada ingrediente.
5. Rastreamento de um lote de ingrediente, permitindo identificar em quais lotes de produção ele foi utilizado.

Esse processo facilita o controle da produção e possibilita localizar rapidamente produtos afetados caso algum lote de ingrediente apresente problemas.
